FunctionJsonDescription
outputraw{clear:boolean,texts:[{content:string,colors:{r:number,g:number,b:number,a:number}}]}输出文字。
elementID 为输出 ID；
clear 为是否清除输出区；
content 为文字；
colors 为颜色；
r 为红色色素；
g 为绿色色素；
b 为蓝色色素；
a 为透明度。
alertraw{texts:{content:string,type:"msg"|"yn"|"input"}}弹出对话框。
content 为文字；
type 为类型。"msg" 表示普通对话框，
"yn" 表示选择对话框，"input" 为输入对话框。
locationchangeraw{pathorurl:string,settings:{newTab:boolean,return:any}}Get to a path or a URL.
pathorurl is the path or the url you want to get into.
newTab is the judge that you open a new tab to have a view or not.
return is the result that returns.
dataimageshowraw{fileset:{dataurl:string,height:number,width:number,border:{style:"dashed"|"double"|"solid",width:number,colors:{r:number,g:number,b:number,a:number}}}}Shows a picture in the result box.
dataurl is the picture URL.
height, width is to set the size of the image.
border.style is set the style of the border. "dashed" is a dozen of square dots. "double" is two lines. "solid" is a single line.
border.width is to set the width of the line of the border.
border.colors is to set the color of the border.
addeventlistenerraw{id:string,event:keyof HTMLElementEventMap,function:Function}This function is to add an event listener to an object or an element.
id is the ID of the element.
event is the event of the element.
function is the event that the event happens of the element.
reloadurlrawReload this URL.
sethtmllanguageraw{language:"ja"|"en"|"ru"|"zh"}Set the URL language.
language is the language to set. "ja" is the Japanese. "en" is the English. "ru" is the Russian. zh is the Chinese.
gethtmllanguagerawGet the result of the URL language.
settitleraw{title:string}Set the URL title.
title is the title that you want to set for the URL.
gettitlerawGet the URL title.
unsettitlerawSet the title as norm.
blankhtmlrawSet the URL blank.
uploadfile2datarawUpload the file with the initialization in the output box.
clearoutputarearawClear the output box.
titleraw{elementID:"output",rawtext:{text:string,color:{r:number,g:number,b:number,a:number},fontSize: number,fontFamily:"DengXian"|"Comic Sans MS"|"Microsoft YaHei"|"inherit"|"汉仪字典体简"|"Arial"|"Arial Black"|"Yu Gothic",onclick?:Function}}让输出的文字更加具体。
fontSize is to set the size of the text.
fontFamily is to set the font of the text.
setwindowframeraw{url:string}Put a URL <iframe> in the output box.
url: the website to show in the box.
geticonrawGet the Icon of the website.
colorchooserrawLet the color frame show.
calculatorraw计算器设置
getrequestraw{param:string}Get the URL request parameter result from location.search.
param is the parameter that you'd like to get.
appendanewlineintheparagraphraw添加空行

copytextraw{text:string}复制文字
appendchildraw{tagset:[{childtag:keyof HTMLElementTagNameMap,properties:[{name:string,value:any}]}]}Append a new HTML tag into output area.
childtag is the tag name that you'd like to append.
properties is the properties that you'd like to set. name is the name of the property. value is the result of the property.
removechildraw{tagname:keyof HTMLElementTagNameMap}Remove an HTML tag from the output area.
tagname is the tag that you'd like to remove.
placefinderraw车牌查询器。
StringCFG.concatraw{before:string,concat:string}在文本后面加上文本。这个指令要用 outputraw 输出。
before 是操作前的文本，
concat 是要加的文本。
StringCFG.replaceraw{before:string,repltxt:string,after:string}Shows the replacement that first showed in the string.
before is the whole text before the replacement.
repltxt is the text that will be replaced.
after is the text that will replace the repltxt.
StringCFG.replaceallraw{before:string,repltxt:string,after:string}Replace all the text that has given.
before is the whole text before the replacement.
repltxt is the text that will be replaced.
after is the text that will replace the repltxt.
StringCFG.deleteraw{before:string,dlttxt:string}Delete a piece of text (can not delete all of them) in a string.
before is the whole text before the deletement.
dlttxt is the text to delete.
StringCFG.$[string1, string2...]快速清屏写文字。
StringCFG._[string1, string2...]快速不清屏写文字。
