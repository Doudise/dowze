阿里谷 (AliCrops) 专栏

说到阿里谷，这个栏目只在 喵妹妹留言板 × 朵梓网 中出现过一次，随后就不了了之了。所以在朵梓网%三%中重现阿里谷，完善代码平台，创建和谐的代码环境。

第一篇 认识篇  前往第二篇
  1. 规定文档格式：
    • HTML：
      <!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Strict//EN" "http://www.w3.org/TR/xhtml1/DTD/xhtml1-strict.dtd">
      其中：
      • html
        指定文档格式为 HTML。必选。
      • PUBLIC "-//W3C//DTD XHTML 1.0 Strict//EN" "http://www.w3.org/TR/xhtml1/DTD/xhtml1-strict.dtd"
        引用 HTML DTD（这个我也不太清楚，但是你只要复制上去就可以了）。可选。
    • XML：
      <?xml version="1.0" encoding="utf-8" standalone="yes" ?>
      其中：
      • version="1.0"
        指定 XML 版本 1.0。必选，值只能是 "1.0"。
      • encoding="utf-8"
        指定 XML 编码。可选。
      • standalone="yes"
        我也不知道这是什么。可选，值只能是 "yes" 或 "no"。
    必须是 HTML 文档的第一行，位于 <html> 标签之前。
  2. HTML5 框架：
    <html>
    <head>
    <meta charset="utf-8" />
    <title>Title</title>
    </head>
    <body>
    文本内容...
    </body>
    </html>
    其中：
    • title
      文档标题，内容可以改变。
    • body
      文章内容。
  3. 规定文本颜色：
    <style>
    i#i {
    color: blue;
    }
    </style>
    <i id="i">这是个蓝色的字。</i>
    效果：
    这是个蓝色的字。
  4. 用按钮改文本内容：
    <script>
    function myFunc1() {
    document.getElementById("id1").innerHTML="示例文本";
    }
    function myFunc2() {
    document.getElementById("id1").innerHTML="样式文本";
    }
    </script>
    <span id="id1">样式文本</span><br />
    <button onclick="myFunc1();">改变文本</button>
    <button onclick="myFunc2();">还原文本</button>
    效果：
    样式文本
  5. 网页提示：
    <script>
    function myFunc3() {
    alert("Alertion");
    }
    </script>
    <button onclick="myFunc3();">提示</button>
    效果：
  6. 网页悬停提示：
    使用<abbr>指令。
    <abbr title="An abbreviation tip">悬停有惊喜</abbr>
    效果：
    悬停有惊喜
  7. 使用表格：
    <table>
    <caption>示例表格</caption>
    <tr>
    <th>标题 1</th>
    <th>标题 2</th>
    </tr>
    <tr>
    <td>文本 1</td>
    <td>文本 2</td>
    </tr>
    </table>
    <style>
    table, th, td, caption {
    border: 1px solid black;
    }
    </style>
    效果：
    示例表格
    标题 1 标题 2
    文本 1 文本 2
  8. 光标不同？
    <style>
    .cursorPointer {
    cursor: pointer;
    }
    </style>
    <u class="cursorPointer">光标不同</u>
    cursor 属性指的是光标。
    效果：
    光标不同
  9. 粗体字放在最中间。
    <b style="text-align: center;">在中间的粗体字</b>
    其中，text-align 指字体位置，left 或 start 指左侧，center 指中间，right 或 end 指右侧。
    效果：
    在中间的粗体字
第二篇 样式篇  返回第一篇
从这一阶段开始，每一个示例都会分为三个部分：CSS、HTML 和 JS 部分。
  1. 圆角矩形：
    CSS 部分：
    div.radioRect {
    border: 2px solid black;
    border-radius: 10px;
    height: 50px;
    width: 100%;
    }
    HTML 部分：
    <div class="radioRect">这是一个圆角矩形</div>
    效果：
    这是一个圆角矩形
    border-radius 意为边框圆角长度，通常用 5px、10px 或者 0.5cm 规定圆角长度。（如果你想要做一个圆形，请看示例 2）
  2. 做一个圆形。
    CSS 部分：
    div.rectcycle {
    border: 2px solid black;
    border-radius: 50px;
    height: 100px;
    width: 100px;
    }
    HTML 部分：
    <div class="rectcycle"></div>
    效果：
    只是这圆里面不能写字，要是能写字就好了。
    原理：CSS 属性 width 和 height 都是 border-radius 值的两倍，这样四条边长度（指直线，不是碰撞面积）都为 0，就是一个圆形了。
  3. 渐变色（从左到右）
    CSS 部分：
    #a1 {
    width:100%;
    height:50px;
    background:linear-gradient(to right, cyan, black, yellow);
    color:white;
    border-radius: 10px;
    text-align: center;
    }
    HTML 部分：
    <div id="a1">渐变色</div>
    效果（由于渐变，首位都会空行）：

    渐变色

    linear-gradient 可接受很多颜色，也可以接受四个方向（及四个延伸方向）或角度（40deg）。
  4. 悬停动画
    CSS 部分：
    #a2{
    transition: 1s;
    background-color: white;
    color: black;
    border: 0.5px solid darkgray;
    }
    #a2:hover{
    transition: 1s;
    background-color: blue;
    color: yellow;
    border: 5px solid black;
    }
    HTML 部分：
    <div id="a2">悬停动画效果：背景变蓝，字体变黄，边框加粗。</div>
    效果（由于渐变，首位都会空行）：

    悬停动画效果：背景变蓝，字体变黄，边框加粗。
